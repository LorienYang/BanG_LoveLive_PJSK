# 全国拉群地图（LoveLive!）

一个基于中国地图的 LoveLive! 同好群聊导航站点。

项目通过地图选省、国外切换、非地区分类、全局搜索、类型筛选和群号复制等方式，帮助用户快速查找各地区或各类型的 LoveLive! 群聊信息。

## 功能特性

- 中国地图交互浏览，支持点击省份查看对应群聊
- 支持国外数据与非地区数据切换
- 支持全局搜索群名、群号与信息字段
- 支持按类型筛选：地区、校群、其他
- 支持排序：默认、认证时间、首字母、类型
- 支持一键复制群号或链接
- 支持亮色 / 暗色模式，并适配系统主题
- 移动端支持底部详情面板与拖拽调整高度

## 项目结构

- [`index.html`](index.html)：页面结构、SEO 信息、资源引入
- [`styles.css`](styles.css)：站点样式、主题变量、响应式布局
- [`app.js`](app.js)：地图渲染、数据请求、交互逻辑、主题控制
- [`china.min.js`](china.min.js)：地图数据脚本
- [`Love_Live.svg`](Love_Live.svg)：站点 Logo
- [`qrcode.webp`](qrcode.webp)：反馈群二维码

## 数据接口

当前使用的接口：

- `https://mapapi.enldm.cyou/api/lovelive`

前端会在 [`app.js`](app.js) 中请求该接口并渲染群聊数据。

## 本地运行

这是一个纯前端静态项目，可以直接使用静态服务器启动。

### 方式一：直接用 VS Code Live Server

将项目目录作为静态站点打开后启动即可。

### 方式二：使用 Python 启动静态服务器

在项目根目录运行：

```bash
python -m http.server 8000
```

然后访问：

```text
http://127.0.0.1:8000/
```

## 页面说明

### 首页地图区

- 点击省份可查看当前省份的群聊信息
- 悬停区域会显示数量提示
- 提供放大、缩小、重置等地图控制按钮

### 详情卡片

- 显示当前省份或当前搜索范围下的群聊列表
- 支持按关键词搜索群名 / 群号
- 支持类型筛选与多种排序方式
- 支持点击复制群号，若识别到链接则可复制链接或直接打开

### 主题模式

项目已适配橙黄色主题，并对深色模式进行了单独处理。

## 适用场景

- LoveLive! 同好社群导航
- 地区群、校群、综合群信息聚合展示
- 静态地图导航类前端页面参考
