# 全国烤群地图

一个面向 **世界计划（Project SEKAI / PJSK）同好** 的全国群聊聚合地图站点。

项目以中国地图为核心展示界面，支持按省份查看群聊分布、搜索筛选、复制群号、深浅色切换，以及移动端适配，适合快速查找地区群、校群和其他相关社群。

## 项目特性

- 基于中国地图展示全国群聊分布
- 支持点击省份查看详细群列表
- 支持全局搜索、分类筛选、排序
- 支持海外与非地区群切换
- 支持一键复制群号或链接
- 支持浅色 / 深色主题
- 支持桌面端与移动端布局
- 使用远程 API 获取最新数据

## 技术栈

- [`index.html`](index.html)：页面结构与 SEO 信息
- [`styles.css`](styles.css)：主题样式、动画、响应式布局
- [`app.js`](app.js)：数据获取、状态管理、地图交互、列表渲染
- [`china.js`](china.js) / [`china.min.js`](china.min.js)：地图绘制依赖
- [`d3@7.9.0`](index.html:224)：SVG 交互与缩放能力

## 数据来源

当前项目仅使用在线接口：

- [`https://mapapi.enldm.cyou/api/pjsk`](app.js:7)

前端会直接请求该接口，并将返回的群数据按省份分组后渲染到地图与详情面板中。

## 运行方式

### 方式一：直接打开

直接用浏览器打开 [`index.html`](index.html) 即可预览页面。

### 方式二：本地静态服务器

如果需要更稳定地测试资源加载与接口请求，建议使用静态服务器：

```bash
python -m http.server 8000
```

或：

```bash
npx serve .
```

## 页面功能概览

### 地图区域

- 展示中国地图
- 根据各省群数量显示不同颜色深浅
- 支持缩放、拖拽、重置视图
- 点击省份查看该地区群详情

### 详情面板

- 显示当前省份群数量
- 支持按关键词搜索群名 / 群号
- 支持按类型筛选
- 支持按时间、名称、类型排序
- 支持复制群号或打开链接

### 其他功能

- 海外群切换
- 非地区群切换
- 反馈二维码弹窗
- 系统主题跟随与手动切换

## 使用场景

- 快速查找自己所在地区的世界计划群
- 统一浏览全国群分布情况
- 帮助社群维护者整理和检查收录情况
- 用于静态部署展示社群导航页面
